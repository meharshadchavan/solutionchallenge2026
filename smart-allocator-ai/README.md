# 🌍 AllocatAI &middot; Open Innovation Prototype 
**Theme Alignment: Smart Resource Allocation**  
**Google Solution Challenge 2026**

AllocatAI is an intelligent resource aggregator and matching engine. It addresses the real-world challenge of inefficiency in supply distributions—whether that is surplus food going to waste while shelters go hungry, or emergency supplies failing to reach disaster zones due to poor logistics visibility.

> **Our Vision:** To build an open innovation platform where municipalities, businesses, and NGOs can pool resource data in real-time. By leveraging Google Gemini 2.0, the platform automatically routes idle resources to critical demands.

---

## 🚀 How It Solves the Challenge Topic
The "[Smart Resource Allocation] Open Innovation" theme asks for solutions that tackle real-world challenges through intelligent distribution logic. AllocatAI achieves this by:
1. **Dynamic Matching:** Using AI to understand complex, unstructured inputs (e.g., "500 boxed meals expiring soon") and matching them to human needs (e.g., "Shelter needs food tonight").
2. **Logistics Engine:** Automatically factoring in available transportation variables to plan movement, not just pairing items.
3. **Open System Architecture:** Designed to take input from any community actor (donor, transporter, or recipient).

---

## 💻 Tech Stack
- **AI Core:** Google Gemini 2.0 Flash API
- **Frontend Framework:** Vanilla HTML/CSS/JS (Lightweight & Accessible)
- **Data Rendering:** Marked.js (Markdown parsing)
- **Deployment:** Ready for Firebase Hosting or Google Cloud Run

## 🏃‍♂️ Getting Started

### Validating the Concept Locally
1. Clone or open the `smart-allocator-ai` folder.
2. Double-click `index.html` to open it in your browser.
3. Obtain your free Gemini API Key from [Google AI Studio](https://aistudio.google.com/apikey) and enter it into the app.
4. Click **"Load Sample Data"** to instantly test the AI with a mock community supply/demand scenario.
5. Click **Generate Allocation Plan** to watch Gemini generate an optimal distribution map and verify alignment with the vision.
